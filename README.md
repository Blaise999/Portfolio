# Blaise-Tech Portfolio

**If you can imagine it, we can build it.**

Production-grade portfolio — Next.js 14 · Tailwind CSS · Framer Motion

## Quick Start

```bash
npm install
npm run dev
# → http://localhost:3000
```

## Deploy

```bash
npx vercel
```

## Sections

1. **Hero** — Brand, tagline, typewriter, stats strip
2. **Services** — 6 service cards (Web Dev, E-Commerce, Smart Contracts, Fintech, Healthcare, Logistics)
3. **Tech Logos** — 16 actual technology logos from CDN (React, Next.js, Solidity, MongoDB, etc.)
4. **Projects** — 6 cinematic project showcases with large images, scrollable galleries, architecture docs
5. **About** — Bio, skill cards, random personality facts
6. **Philosophy** — Terminal-style manifesto with engineering principles
7. **FAQ** — 8-question accordion ("What Our Clients Want to Know")
8. **Contact** — CTA banner, WhatsApp + Email buttons, 6-platform social grid
9. **WhatsApp Widget** — Floating chat button with popup, green pulse ring
10. **Footer** — Logo, tagline, nav links
